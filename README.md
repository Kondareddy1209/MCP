# MCP Server

This repository contains the MCP (Mobile Control Protocol) server implementation.

## Overview
- Provides backend services for mobile device management.
- Includes API endpoints, analytics, and notification dispatch.
- Built with Python and a lightweight frontend.

## Getting Started
1. Clone the repository.
2. Install dependencies: pip install -r requirements.txt.
3. Run the server: python mcp_server.py.
